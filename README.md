# LeatherArmorPicker
by Command Macro in Minecraft 1.20.2
浴衣はこっち→https://github.com/CKenJa/YukataTrim

## How to use

まずセットアップを行う。地面の下にコマンドブロック埋め込んで
`function ckenja.yukata:pos_picker/init`
を実行

リピートコマンドブロックで同座標で
`function ckenja.yukata:color_display/wrapper`
を実行。
